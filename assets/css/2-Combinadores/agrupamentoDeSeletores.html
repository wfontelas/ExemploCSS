Agrupamento de Seletores

Podemos aplicar as mesmas regras de CSS para vários seletores diferentes.Para isso, podemos separar os eletores utilizando uma vírgula(,).

Assim, não preisamos ficar copiando os mesmos códigos para todos , nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras!
