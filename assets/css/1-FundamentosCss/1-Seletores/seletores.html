Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML epecifíca
  • Seletor por ID(#):Busca elementos através do atributo "Id"
  • Seletor por Classe(.):Busca elementos através do atributo "class"
  • Seletor Universal(*):Seleciona todos os elementos HTML
  • Seletor de Atributo([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com valor especifico
